Bem Vindo!

 

 

Projeto Web Fullstack completo com tecnologias como Spring Boot 2.x, Angular 10 e 11, Heroku, JPA, Git, Github, MySQL, H2

O Site fornece um gerenciamento de livros simples, mas com várias boas práticas em código que facilitam a manutenção e adição de novas funcionalidades.

Podemos listar, criar, editar e excluir livros e categorias inclusive com responsividade adaptado a telas menores de smartphone.

Usamos Validação tanto do backend quanto com formulários reativos do Angular Material.

 

Toda a aplicação é organizada em commits sobre cada parte do projeto.
